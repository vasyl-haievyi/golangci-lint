Important

testifylint has breaking changes about enabling/disabling checks:

  • If you were using the option enable with a filtered list of checks, you should either add disable-all: true (1) or use disable field (2).

    # Example (1)
      testifylint:
        disable-all: true
        enable:
          - bool-compare
          - compares
          - empty
          - error-is-as
          - error-nil
          - expected-actual
          - go-require
          - float-compare
          - len
          - nil-compare
          - require-error
          # - suite-dont-use-pkg
          - suite-extra-assert-call
          - suite-thelper
    # Example (2)
      testifylint:
        disable:
          - suite-dont-use-pkg
